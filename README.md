# GAG2ServerPetChecker
This script gives you 0 game advantage. It was made for people like me who like to AFK Grow a Garden 2. This script sends pet model names from the wild pet spawns to the selected webhook instantly. 


how to use?:
inject script -> you see menu with status and a tab where you put your webhook. -> you put webhook -> test to see if it is working -> if it is working than just lay back and relax while the script does its job
